#Prime factorization

Simple Programe that finds prime numbers and test if it is a factor of the given input... 
It then forms the table just the way we make in notebooks and then you can copy it in note book as it is...

eg

input : 8

output : 
2 | 8
\-----
2 | 4
\-----
2 | 2
\-----
1 |
\-----

2 X 2 X 2 = 8

Its simple ;-)
